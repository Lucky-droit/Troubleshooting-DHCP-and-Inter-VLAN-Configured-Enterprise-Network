# Troubleshooting DHCP and Inter-VLAN Configured Enterprise Network

## Project Overview
This project demonstrates the implementation and troubleshooting of an enterprise network using VLAN segmentation, inter-VLAN routing (router-on-a-stick), and centralized DHCP services.

## Technologies Used
- Cisco Packet Tracer
- VLANs and 802.1Q Trunking
- Inter-VLAN Routing
- DHCP Configuration
- Cisco IOS Troubleshooting Commands

## Key Troubleshooting Scenarios
- DHCP clients not receiving IP addresses
- VLAN and access port misconfiguration
- Trunk port issues
- Missing or incorrect router subinterfaces
- Inter-VLAN connectivity failures

## How to Use
1. Download the `.pkt` file
2. Open it in Cisco Packet Tracer
3. Review configurations and simulate troubleshooting scenarios
